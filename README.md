## 📌 Project Overview
Comprehensive lab mastering **5 prompt engineering techniques** and **6 LangChain application patterns** using IBM Watsonx. Covers in-context learning strategies from basic prompts to Chain-of-Thought, plus LangChain PromptTemplates for real-world tasks.

**Domain:** Prompt Engineering + LangChain  
**LLM:** IBM Watsonx  

## 🔑 5 Prompt Engineering Techniques

| Technique | Description |
|---|---|
| **Basic prompt** | Simple direct text input |
| **Zero-shot** | Task with no examples |
| **One-shot** | Task with 1 example |
| **Few-shot** | Task with multiple examples |
| **Chain-of-Thought (CoT)** | Step-by-step reasoning |
| **Self-consistency** | Multiple reasoning paths → best answer |

## 🛠️ 6 LangChain Application Patterns
- **PromptTemplate** — variable injection
- **Text summarization** — document summarizer
- **Question answering** — grounded Q&A
- **Text classification** — sentiment/category
- **Code generation** — natural language → code
- **Role playing** — persona-based responses
- **LCEL Chain** — `prompt | llm | parser`
